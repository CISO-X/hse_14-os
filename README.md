# Automated Threat Monitoring and Response

Учебный проект по итоговому заданию по Python в области информационной безопасности.

## Описание

Скрипт автоматически собирает и анализирует данные из двух источников:

- **логи Suricata**
- **данные об уязвимостях Vulners** (или демо-данные при отсутствии доступа к API)

На основе анализа выполняется простое реагирование:

- имитация блокировки подозрительных IP-адресов и доменов
- уведомление о критичных уязвимостях

Дополнительно формируются:

- **CSV-отчёт** со списком найденных угроз
- **PNG-график** по источникам угроз

---

## Функциональность

- анализ alert-событий Suricata
- выявление аномально частых DNS-запросов
- обработка данных об уязвимостях
- генерация CSV-отчёта
- построение графика
- вывод действий реагирования в консоль
